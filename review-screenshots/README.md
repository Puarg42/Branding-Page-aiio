# Legacy Review Screenshots

This folder is retained only so older generated screenshots stay ignored locally.

The current release review system now writes to:

```text
review/
```

Use the current command:

```bash
npm run review
```

The generated package contains desktop and mobile section screenshots, `review/index.html`, `review/manifest.json` and `review/README.md`.

The legacy `review-screenshots/` output is no longer the standard review destination.
