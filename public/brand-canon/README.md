# aiio Brand Canon Asset Governance

The Brand Canon is treated as an official design system.

## Approved assets

`public/brand-canon/`

Only approved Brand Canon assets live at this level. Website code may reference these files directly.

## Incoming assets

`public/brand-canon/_incoming/`

Work in progress only. Website code must never reference files from this folder.

## Archived assets

`public/brand-canon/_archive/`

Rejected or historical artwork only. Website code must never reference files from this folder.

## Validation

Before an illustration is referenced by the website, it must match the approved Brand Canon in abstraction level, geometry, perspective, material language, lighting, layer language, color palette, editorial quality, visual calmness and composition.

Reject images that feel like generic AI artwork, cyberpunk visuals, dashboards, robots, holograms, circuit boards, stock imagery or technology marketing.
