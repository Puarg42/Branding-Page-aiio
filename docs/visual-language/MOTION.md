# Motion

## Purpose

Define how motion supports the aiio visual language without distracting from the idea.

## Principle

Motion communicates understanding.

It does not communicate energy.

It does not communicate technology.

## Allowed Motion

1. Slow light flow
2. Calm opacity changes
3. Subtle translateY

## Prohibited Motion

1. Blur
2. Rotation
3. Bounce
4. Decorative floating
5. Explosive glow
6. Spectacle parallax
