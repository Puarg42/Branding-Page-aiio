# SPRINT 007 - Website Narrative Review

## Objective

Document the current narrative, content, imagery, section roles, entry points and site inventory before the next design or implementation sprint.

## Review Scope

- Homepage narrative
- Website information architecture
- Section roles
- Content density
- Redundancy
- Image usage
- Brand Canon alignment
- Entry points and user journeys
- Existing visible routes

## Review Documents

- `01-narrative-map.md`
- `02-content-audit.md`
- `03-redundancy-audit.md`
- `04-image-audit.md`
- `05-section-roles.md`
- `06-entry-point-audit.md`
- `07-site-inventory.md`

## Method

1. Review the live homepage in sequence.
2. Identify the role of each section.
3. Check whether the narrative progresses without repetition.
4. Compare image usage against the aiio Brand Canon.
5. Check whether each page has one clear purpose.
6. Document gaps before making further changes.
