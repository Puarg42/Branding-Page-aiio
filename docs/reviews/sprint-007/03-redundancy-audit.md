# 03 - Redundancy Audit

## Purpose

Identify repeated arguments, duplicated hierarchy, repeated wording and sections that make the same point.

## Redundancy Criteria

- Same idea appears in multiple sections
- Same progression appears more than once
- Same phrase is repeated without strategic intent
- A section explains what the previous section already made clear
- A visual repeats a concept already established by text
