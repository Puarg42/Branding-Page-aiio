# Sprint 008.1 - Executive Copy Polish

## Objectives

- Elevate visible headlines, subheadlines and CTAs to executive-level quality.
- Preserve layout, page structure, Brand Canon assets and visual system.
- Reduce generic SaaS wording and clarify section ownership.

## Headlines Changed

- `How organizations become intelligent.` -> `From knowledge to capability.`
- `The system behind the journey.` -> `Organizational Intelligence needs an operating system.`
- `The next generation will be defined by how quickly it learns. And how reliably it turns learning into capability.` -> `The next generation turns learning into capability.`
- `How Organizational Intelligence becomes operational.` -> `Where Organizational Intelligence becomes operational.`
- `One platform. Four capabilities.` -> `Four capabilities. One system.`
- `Why this shift is inevitable.` -> `A new discipline needs a theory.`
- `The defining innovation of the AI era is not that machines become intelligent.` -> `The breakthrough is organizational understanding.`
- `The same architecture guides every visual theory.` -> `The visual system follows the theory.`
- `Build the capability era with aiio.` -> `Help organizations build lasting capability.`
- `We are helping define the next generation of organizational management.` -> `Organizations need understanding that lasts.`
- `What guides aiio.` -> `Principles guide the work.`
- `Start the conversation.` -> `Start with the right conversation.`

## Subheadlines Changed

- Platform page intro shortened and reframed around one continuous system.
- Thinking core thesis support shortened.
- Partners intro shortened and made role-specific.
- Company mission body shortened.
- Contact intro and contact panel copy clarified.

## CTAs Changed

- `Explore OIS` -> `See the system`
- `Explore ProcessCollector` -> `See ProcessCollector`
- `Explore ProcessMagnet` -> `See ProcessMagnet`
- `Explore ProcessForge` -> `See ProcessForge`
- `Explore DataForge` -> `See DataForge`
- `Explore the Platform` -> `See the Platform`

## Terminology Decisions

- Use `Organizational Intelligence` as the category term.
- Use `understanding` for the missing layer and semantic shift.
- Use `capability` for observable organizational outcome.
- Avoid `OIS` in primary CTAs.
- Keep `Platform` as navigation and page context, but do not make it the lead idea.

## Open Editorial Questions

- Whether the homepage still needs both `Imagine` and `CEO Moment` in the long-term V1 narrative.
- Whether `Self-Enabling Organization` should remain as a homepage concept or move deeper into Thinking.
- Whether the Platform page should later show clearer enterprise integration language for CIO and Enterprise Architect audiences.

## Recommendations for Sprint 009

- Define category architecture before adding more website copy.
- Clarify the relationship between `Organizational Intelligence`, `Organizational Understanding` and `Capability`.
- Preserve the executive tone: fewer claims, sharper structure.
- Avoid adding more CTAs until page roles are fully settled.
